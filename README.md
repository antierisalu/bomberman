# bomberman-dom

The Bomberman Game

# How to run Dev environment:
> <br/>
>
>>
>>#### 1) Clone the repo:
>>```bash
>>git clone https://01.kood.tech/git/claks/bomberman-dom
>>```
>
>>#### 2) Navigate to the project root dir:
>>```bash
>>cd bomberman-dom/
>>```
>
>>#### 3) Install npm modules with init script:
>>```bash
>>npm run init
>>```
>
>>#### 4) Run the Backend+Frontend with npm:
>>```bash
>>npm run bomber
>>```
